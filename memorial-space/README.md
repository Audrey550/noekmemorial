# Noek Memorial

A digital memorial space where family and friends can create and visit personalized remembrance rooms.

## Features

- Create memorial rooms
- Public and private rooms
- Invite-code access
- Co-editor invitations
- Room customization
- 3D memorial environment
- Profile system
- Supabase cloud storage

## User Roles

### Admin
- Create rooms
- Manage room settings
- Invite co-editors
- Generate invite codes
- Edit room content

### Editor
- Access invited rooms
- Edit room content
- Contribute to memorial spaces

### Viewer
- View memorial content

## Tech Stack

- Vue 3
- Vite
- Supabase
- Three.js

## Installation

```bash
npm install
npm run dev
```

## Database

This project uses Supabase.

Required database setup can be found in:

```text
supabase_room_setup.sql
```
